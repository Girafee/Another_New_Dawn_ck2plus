Another New Dawn: ck2plus version
Requirements
- Crusader Kings II patch 2.0.4
- CK2+ 2.07.0
- (optional) Titular Title Generator 2.0.18


Installation:
1. Copy the base files from CK2+ to the documents/PI/CKII/mod folder. Submods are untested but could work alongside.
2. Copy the AND_PLUS/KAforMultipleMods folders and .mod files to the C:\Users\<yourusername>\Documents\Paradox Interactive\Crusader Kings II\mod folder.
!!!Remember to always take the 'Redetermine de jure lieges' when loading a saved game!!!

(Optional!) Installation with new KA files
1. Copy the base files from CK2+ to the documents/PI/CKII/mod folder. Submods are untested but could work alongside.
2. Copy the AND_PLUS folder and .mod file to the C:\Users\<yourusername>\Documents\Paradox Interactive\Crusader Kings II\mod folder.
3. Launch the game once and exit. This is necessary for the generator to attain the correct settings from ck2plus.
4. Extract the titular generator
5. Copy the contents of the titlegen folder (Another_New_Dawn_ck2plus/info/titlegen files) to the titlegen/scripts folder and let them overwrite the contents.
6. Launch the generator - > Check the NivveAllows, NivveGains and NivveRuleset scripts (no other scripts checked!)
5. Check both Another New Dawn and the ck2plus mod and click 'Load Data'
6. Go to the next tab and check the options how you prefer them. The default options are mentioned below.
7. After generating the files you're ready to play.
!!!Remember to always take the 'Redetermine de jure lieges' when loading a saved game!!!

*default settings:
Short names: empires
Title creation: all
restrictions: 3/3/3
replace de jure on creation: yes
History: Generation - Full history - 867
History options: 
- realms 1040/1040
make kingdoms: unchecked
- Republics 15/15
- Theocracies 50/50
